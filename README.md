# Aethering — 3D 오픈월드 (프로토타입)

Three.js로 만든 **3인칭 3D 오픈월드** 워킹 프로토타입입니다. 프로시저럴 지형 위를 캐릭터로
걷고·달리고·점프하고·공격합니다. "오픈월드 MMORPG"로 가는 **1단계(걸어다니는 월드)** 토대입니다.

> ⚠️ **로컬 서버로 열어야 합니다.** ES module + importmap + GLB 로드라 `index.html` 더블클릭(file://)으로는
> 동작하지 않습니다. 아래처럼 정적 서버로 여세요.

## 실행

```bash
cd /Users/jun_aay/workspace
python3 -m http.server 8123
# 브라우저에서:  http://localhost:8123/mmo/
```
(편의 스크립트: `mmo/serve.command` 더블클릭 → 서버 실행 + 브라우저 열기)

## 게임 흐름 (던전 로그라이크)

**캐릭터 선택**(기사/마법사/궁수 — 공격·스킬 차등) → **지하 1층** 입장 → 적과 **보스(해골 군주)** 처치 →
**보스를 잡으면 다음 층**으로. 층마다 **바이옴(테마)이 통째로 바뀝니다**: 🌲숲 → 🦇동굴 → ❄️설원 → 🌋용암 동굴 (순환).
지형 텍스처·식생·하늘·조명·안개가 전부 교체되고(동굴/용암은 어두워 플레이어 주변 광원·발광 크리스탈), 깊을수록 적이 강해집니다.
클리어 시 전체 회복, 처치로 XP·골드·레벨업.

**적 종류**: 해골 병사(기본)·전사(탱커)·도적(빠름)·**술사(원거리)**. **보스는 층마다 순환하며 고유 패턴**:
해골 군주(범위 슬램)·대마법사(투사체 난사)·사신(돌진)·강령술사(소환).
**폐쇄형 바이옴(동굴/용암)**은 돌벽·천장으로 둘러싸여 하늘이 안 보이고, 벽·기둥에 막힙니다.

**조준**: 공격 시 가까운 적을 **자동 조준**(소프트 락온)해 캐릭터가 그쪽을 보고 발사합니다.
**보스 이동기**: 군주=도약(점프 돌진+착지 슬램), 대마법사/강령술사=블링크(순간이동), 사신=돌진.
**레벨업 스킬 드래프트**: 레벨업마다 **스탯 스킬 3장 중 1택**(화력·연사·치명타·다중사격·관통·흡혈·폭발확장·신속·강철체력·스킬연마)으로 빌드업.
**액티브 스킬(보스 보상)**: 층 보스를 잡으면 **티어별(실버/골드/프리즘) 액티브 스킬**을 1개 획득해 **1·2·3·4 슬롯(최대 4개)**에 장착·교체(전진키 W와 겹치지 않게 숫자키 사용). 풀은 **직업당 30종(티어별 10종, 총 90종)**. 단순 광역뿐 아니라 **지속 피해 장판·생명 흡수·중력장(끌어당김)·넉백·돌진 강타·시한폭탄·처형·자기 버프(공속/피해/치명/방어)·부채꼴 난사·산탄** 등 다양한 메커니즘. **이동+공격형**: 궁수 **갈고리 이동사격**(앵커로 비행하며 화살 난사, 동굴은 천장에 고정)·**강습 사격**, 기사 **검무**(적 사이를 무적으로 누비며 시점이 함께 도는 연속 난무). 궁수 스킬에는 마법진 연출이 빠집니다(직업 컨셉). 치유술·화염폭발·빙결·연쇄번개·수호방패·대지강타·메테오·시간정지·광폭화 등. 클래스 스킬(패링/구르기/블링크)은 **우클릭**.
투사체는 **3D 자동조준**(점프 중에도 적을 향해 위/아래로 발사).
**직업별 스킬 풀**: 레벨업 스탯·보스 보상 액티브 모두 직업(기사/마법사/궁수)마다 풀이 완전히 분리됩니다. 레벨업 드래프트는 **공용 3종 + 직업 전용 약 30종(총 95종)**이며, 매 드래프트에 직업 전용 스킬이 1~2장 반드시 포함됩니다. 보스 보상 액티브도 **전부 직업 전용**(공용 없음): 기사=방패강타/대지강타/광폭화/불멸…, 마법사=빙결/연쇄번개/메테오/시간정지…, 궁수=화살세례/맹독살포/강철폭풍….
**무빙기(클래스 스킬)**: 기사=패링·마법사=블링크·궁수=구르기. **`F` 키 또는 우클릭**으로 사용(액티브 1·2·3·4와 별개).
**티어 희귀도**: 보스 보상은 실버<골드<프리즘 가중치 랜덤 — 좋은 등급일수록 덜 나오고, 깊은 층일수록 상위 티어 확률↑.
**연출/타격감**: UnrealBloom 발광 후처리(바이옴별 강도), 적 피격 흰 플래시, 크리·폭발 시 히트스톱.
**VFX(스프라이트 파티클)**: 가산혼합 글로우 텍스처(캔버스 생성) 기반으로 — 화살/파이어볼/적 볼트에 **발광 코어+잔광 트레일+머즐 플래시**, 폭발은 **섬광 코어+확장 충격파 링+연기(smoke.png)+스파크**, 연쇄번개는 이중 라인+착탄 스파크, 메테오는 화염 꼬리. (연기 텍스처는 three.js 예제 CC0/MIT, 나머지 글로우/링/스파크는 런타임 캔버스 생성)
**원소 스킬 VFX**: 빙결=얼음 폭발(청록 섬광+얼음 파편 솟구침+서리 모트), 눈보라=광역 연속 서리 폭풍, **아이스 토네이도/태풍=회전 입자 기둥(중심 코어)**, 시간 정지=화면 암전+회전 시계 링+적 정지 입자, 화염/맹독 폭발=원소색 섬광+링+(독)연기, 보호막/광폭화/치유=플레이어 추적 회전 오라. (이미터·얼음 파편 시스템 신규)
**애니메이션 스프라이트시트**: 실제 다운로드 에셋(Phaser explosion.png, 64px×23프레임)으로 **폭발·메테오·연쇄번개 착탄·원소 폭발**에 진짜 화염 애니메이션 재생(원소 색 틴트). 바닥 룬 마법진·상승 잉걸불·화면 색 섬광·흰열 폭심도 함께.
**블링크**: 이동 입력(WASD) 방향 우선, 없으면 시선 방향으로 순간이동(적에게 끌려가지 않음).

> 개발용: `?auto=knight|mage|archer` 로 선택 건너뛰기, `&floor=N` 특정 층, `&draft=1` 레벨업 드래프트 테스트.

| 동작 | 키 |
|------|----|
| 이동 | `W A S D` / 방향키 |
| 달리기 | `Shift` |
| 점프 | `Space` |
| 공격 | 마우스 좌클릭 |
| 액티브 스킬 (4슬롯) | `1` `2` `3` `4` |
| 클래스 스킬 (구르기/패링/블링크) | `F` 또는 마우스 우클릭 |
| 시점 회전 | 마우스 드래그 (이동 중엔 진행 방향으로 자동 추적) |
| 줌 | 마우스 휠 |

> 개발용: `?auto=knight|mage|rogue` 를 URL에 붙이면 선택 화면을 건너뛰고 바로 시작합니다.

## 현재 구현

**Phase 1 — 월드/이동**
- 프로시저럴 지형(높이맵·버텍스 컬러 로우폴리), 물, 그림자, 하늘/안개
- 나무·바위 프로시저럴 산포(수백 개)
- CC0 애니메이션 캐릭터 + 상태머신(Idle/Walking/Running/Jump/Punch)
- 진행 방향 회전, 중력·점프·지형 접지, 3인칭 추적 카메라(드래그 회전·휠 줌), HUD

**Phase 2 — 전투/퀘스트/NPC** (신규)
- **몬스터**: 그림자 병사·정예 그림자(배회→추격→공격 AI, 체력바, 처치 시 8초 뒤 부활)
- **전투**: 좌클릭 펀치(정면 범위 판정·크리티컬), 몬스터 반격, 플레이어 HP·사망·부활, 떠오르는 데미지 숫자
- **NPC**: 마을 장로(퀘스트 부여)·경비병, `E`로 대화(말풍선 UI)
- **퀘스트**: 장로와 대화 → 그림자 5처치 → 정예 3처치(체인), 우상단 퀘스트 추적기
- **성장**: 처치 시 XP·골드, 레벨업(최대 체력·공격력 증가), HP/XP/Lv/골드 HUD
